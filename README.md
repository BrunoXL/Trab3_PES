# Trabalho 3 - Biblioteca Digital (Lua) 

O trabalho consiste em sistema para web em Lua que guarda artigos ciêntíficos e os ordena através do número de citações de palavras-chave.

## Planejamento
Em nosso grupo dividimos que em cada uma das entregas teremos um gerente de projeto, para dar experiência para todos do grupo, e cada tarefa será dividida por esse gerente em cada uma das entregas em separado. Para cada ma das tarefas teremos sempre metade do grupo responsável e a outra realizando a revisão dessa.

## Entregas
